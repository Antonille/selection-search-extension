Selection Search Launcher

Edit the config JSON, then click Save. The right-click menu updates automatically.

Accepted config formats

  • A simple top-level array of providers using urlTemplate and optional filters.
  • A richer object with finalists, like your URL-template knowledge base file.

Menu grouping

The extension groups entries by category first, then creates a submenu for repeated site families or domains. That keeps the right-click menu from becoming a long flat list.

Simple provider-array example

[
  {
    "id": "google",
    "title": "Google",
    "enabled": true,
    "openInBackground": false,
    "urlTemplate": "https://www.google.com/search?q={text}",
    "filters": {
      "trim": true,
      "collapseWhitespace": true,
      "encoding": "url",
      "regexReplacements": []
    }
  }
]
