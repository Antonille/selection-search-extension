Selection Search Launcher

The packaged providers.default.json file is the live default. Save or import only when you want a local override.

Accepted config formats

  • A simple top-level array of providers using urlTemplate and optional filters.
  • A richer object with finalists.
  • The new spec object with a top-level resources array and explicit placeholder pipelines.

Why the override model changed

In older revisions, the packaged JSON was copied into extension storage once and then silently diverged from the file on disk. In this revision, editing providers.default.json in the unpacked folder and clicking Reload in chrome://extensions immediately changes the live default behavior.

Menu grouping

The extension groups entries by category first, then by site family. The new spec can also explicitly set menu.category_label and menu.group_label.
